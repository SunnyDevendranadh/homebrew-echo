# homebrew-echo

Homebrew tap for [Echo](https://github.com/SunnyDevendranadh/Echo) — a Rust-native AI coding agent (BYOK, always-on kernel sandbox, one static binary).

```bash
brew install sunnydevendranadh/echo/echo
```

Formula installs the prebuilt macOS arm64 release binary (sha256-pinned). Other platforms: build from source (`cargo build --release`).
